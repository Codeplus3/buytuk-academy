## الوثائق

| الوثيقة | الغرض | يقرأها |
|---------|-------|--------|
| **Business_Model.md** | من يدفع لمن مقابل ماذا ولماذا يستمر | المؤسسون + المستثمرون |
| **Pricing_Model.md** | الخطط والأسعار وسياسة التسعير | المبيعات + الإدارة |
| **Go_To_Market.md** | كيف نصل لأول 100 مدرسة | المبيعات + التسويق |
| **Pilot_Strategy.md** | كيف نختار ونُدير ونُحوّل الـ Pilot | المبيعات + Customer Success |
| **Customer_Onboarding.md** | رحلة العميل من التوقيع للاستخدام الفعلي | Customer Success |
| **Institution_Onboarding.md** | الإعداد التقني الداخلي لـ Tenant جديد | فريق العمليات |
| **Support_SLA.md** | ما نعد به وما نلتزم به لكل خطة | الدعم + المبيعات + العملاء |
| **Sales_Playbook.md** | كيف تبيع BuyTuk من اللقاء الأول للتوقيع | فريق المبيعات |
