## المراحل الثلاث

### المرحلة 0 → 1: إثبات المفهوم (Phase T-0 إلى B-0)
```
الهدف    : مدرسة واحدة تدفع وتستخدم وتُثبت القيمة
النهج    : Sales مباشر شخصي — بانيو المشروع يبيع بنفسه
الزمن   : 3-4 أشهر بعد إتمام Phase T-0

الإجراءات:
  □ حدّد 5 مدارس خاصة في دائرتك الشخصية (warm network)
  □ اطلب اجتماعاً لعرض النظام (30 دقيقة)
  □ قدّم Demo حياً مع بيانات وهمية جذابة
  □ اقترح Pilot بسعر رمزي
  □ وقّع واحدة وابدأ (Pilot_Plan.md)

النجاح: مدرسة واحدة مدفوعة تستخدم النظام أسبوعياً
```

### المرحلة 1 → 10: التحقق من السوق (Phase B-0 إلى B-1)
```
الهدف    : 10 مدارس مدفوعة بحلول الشهر 12
النهج    : Word of Mouth + شبكة التعليم الخاصة
الزمن   : 6-12 شهراً بعد نجاح Pilot الأول

قنوات النمو:
  1. Referrals من Pilot الأولى
     → مدير المدرسة يُعرّف مدير مدرسة أخرى
     → حافز: خصم 10% للمُحيل ولأول شهر للمدرسة الجديدة

  2. مجتمعات التعليم
     → مجموعات واتساب ومنتديات مديري المدارس
     → حضور ملتقيات تعليمية محلية
     → مشاركة حالة نجاح Pilot بإذن المدرسة

  3. Direct Outreach
     → قائمة المدارس الخاصة في المنطقة
     → ابدأ بالمدارس التي تدفع على الجودة (رسوم عالية = مستعدون للاستثمار)

المقياس: معدل إغلاق (Close Rate) اجتماعات العرض > 30%
```

### المرحلة 10 → 100: التوسع (Phase B-2 إلى E-0)
```
الهدف    : 100 مدرسة بحلول السنة الثالثة
النهج    : Channel Partners + Inbound + Enterprise

قنوات النمو:
  1. Channel Partners
     → شراكات مع موردي تقنية التعليم الموجودين
     → موزعون إقليميون يبيعون BuyTuk بعمولة
     → ناشرو كتب مدرسية يُضيفون BuyTuk كخدمة مكملة

  2. Inbound Marketing
     → محتوى تعليمي: "كيف تكتشف فجوات تعلم الطلاب"
     → Case Studies من مدارس ناجحة (بإذنها)
     → SEO: أسئلة مديري المدارس والمعلمين

  3. Enterprise Deals
     → وزارة تعليم أو إدارة تعليمية → عقد يغطي 10-50 مدرسة دفعة واحدة
     → يحتاج: Legal team، SLA رسمي، procurement process
     → الزمن: أطول (6-18 شهر للإغلاق) لكن القيمة أكبر

  4. Conferences
     → عرض في مؤتمرات التعليم الإقليمية
     → جائزة الابتكار التعليمي → PR مجاني
```

## الرسالة التسويقية (Messaging)

### للمعلم
```
العنوان: "اعرف بالضبط لماذا يتأخر طالبك — واعرف ماذا تفعل"
التفصيل: بدلاً من "يحتاج تحسيناً في القراءة" — تعرف:
          "ضعف في الفهم العميق — ليس في الطلاقة"
          وتحصل على خطة علاج جاهزة في دقائق
CTA: "جرّب مع صفك مجاناً لأسبوعين"
```

### للمدير
```
العنوان: "صورة المدرسة كاملة — لا تنتظر التقييم النهائي"
التفصيل: لأول مرة: ترى فجوات الطلاب لحظة حدوثها
          وتعرف أي الفصول تحتاج تدخلاً
          وعندك التقرير جاهزاً للوزارة
CTA: "اطلب Demo مدته 30 دقيقة"
```

### للمستثمر / الشريك
```
العنوان: "المنصة الوحيدة التي تقيس أثر التعلم — لا درجاته"
التفصيل: 6 محركات متكاملة + خط تشخيص → علاج → أثر
          موثّق بـ 20+ وثيقة + كود جاهز للـ Enterprise
CTA: "اطلع على المشروع"
```

## المقاييس الأساسية (GTM Metrics)

```
Pipeline Metrics:
  Leads / شهر          → عدد المدارس التي دخلت في محادثة
  Demo Conversion       → نسبة Leads → Demo
  Close Rate            → نسبة Demo → Pilot
  Pilot-to-Paid Rate    → نسبة Pilot → اشتراك حقيقي
  الهدف: Lead→Paid > 15%

Revenue Metrics:
  MRR (Monthly Recurring Revenue)
  ARR (Annual Recurring Revenue)
  Average Deal Size
  Time to Close (أيام من أول اتصال لتوقيع العقد)

Retention Metrics:
  Annual Churn Rate (الهدف < 10%)
  Net Revenue Retention (الهدف > 100% — توسع داخل العميل)
  NPS Score (الهدف > 50)
```
