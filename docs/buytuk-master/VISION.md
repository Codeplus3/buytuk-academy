## القيم الجوهرية (Values)

### V-01: الأدلة أولاً
```
لا حكم على طالب بدون دليل موضوعي مُجمَّع.
الدرجة وحدها لا تكفي — نريد السبب.
```

### V-02: AI مساعد لا حاكم
```
AI يقترح التشخيص والعلاج.
الإنسان (المعلم) يقرر دائماً.
لا قرار يُلزم الطالب بدون مراجعة بشرية.
```

### V-03: الشفافية الكاملة
```
الطالب يعرف لماذا حصل على هذه النتيجة.
ولي الأمر يعرف لماذا هذا العلاج.
المعلم يعرف كيف حسب النظام الدرجة.
المنطق لا يختبئ في "صندوق أسود".
```

### V-04: الخصوصية بالتصميم
```
بيانات الطفل ملك للطفل وأسرته.
لا مشاركة بيانات شخصية بدون موافقة صريحة.
البيانات المُجمَّعة مُجهَّلة دائماً.
```

### V-05: البساطة تقود القيمة
```
الميزة المعقدة التي لا يستخدمها المعلم = صفر.
كل إضافة يُسأل: هل تحل مشكلة حقيقية؟
```

### V-06: الاستمرارية المؤسسية
```
المشروع يعيش بعد الفريق الحالي.
كل قرار يُوثَّق.
كل معرفة تُحفظ.
```
